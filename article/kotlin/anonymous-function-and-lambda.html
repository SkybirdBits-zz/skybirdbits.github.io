بررسی یه ویژگی از کاتلین:

در کاتلین همه چیز دارای مقدار است؛ از متغیر ها گرفته تا توابع تقریبا هر عبارت و تابعی که وجود داره دارای مقداره.

فرض کنید تو جاوا تابعی رو صدا میزنیم اگه مقداریو برگردونه میتونیم به متغیر اختصاص بدیم: 

            
    private String myMethod(){
        return "This is a result type of String"
    }

    //correct
    String result = myMethod();

اگه مقداریو بر نگردونه چیزی رو نمیتونیم به متغیر اختصاص بدیم و دچار خطای کامپایل میشیم: 

            
    private void myMethod(){
        ...
    }

    //compile error
    String result = myMethod()

در کاتلین void هم یک مقدار داره از نوع Unit :

            
    private fun myMethod(){
    ...
    }

    //correct
    val result = myMethod()

عبارت لامبدا (lambda-expression) در کاتلین:

برای تعریف متغیر هایی از جنس لامبدا مانند متغیر های معمولی عمل میکنیم:

اسم متغیر: اسم دلخواهیه که برای متغیر در نظر میگیریم

نوع متغیر: همان نوع لانداییه که میخوایم تعریف کنیم در داخل () پارامتر هایی که میخوایم عبارت داشته باشه رو تعریف میکنیم و Type همان نوعیست که عبارت بر میگردونه

مقدار لامبدا: همان مقدار متغیر است و با {...} متغیرو‌ مقداردهی میکنیم

توجه: اگه بخوایم عبارت پارامتر داشته باشه به صورت زیر براش پارامتر تعریف میکنیم: 

            
    val myLambda: (T0, T1 , ... , Tn) -> Type = {
        t0, t1, ... tn ->
        ...
        type
    }

در بالا T0 , T1 ... Tn پارامتر هاییست که می‌خوایم عبارت داشته باشه و t0, t1, ...,tn اسامی دلخواهیه که به پارامتر ها اختصاص میدیم و داخل {} از پارامتر ها استفاده میکنیم، type متغیری از جنس Type است که برگردونده میشه

سپس عبارتو‌ مانند تابع معمولی به صورت زیر صدا می‌زنیم: 

            
    val t0 = T0()
    val t1 = T1()
    ...
    val tn = Tn()
    val type = myLambda(t0, t1, ..., tn)

مثال: 

            
    val myLambda: () -> String = {
        "Hello From Lambda"
    }

در مثال بالا عبارت لاندا بدون پارامتر است و مقدار "Hello From Lambda" رو پس از اجرا برمیگردونه

صدا زدن عبارت به صورت تابع: 

            
val result: String = myLambda()

مثال: 

            
    val myLambda: (String) -> String = {
        it -> "Hello $it, From Lambda!"
    }

در مثال بالا عبارت لاندا داری یک پارامتر است از جنس String

صدا زدن عبارت به صورت تابع: 

            
    val name = "Joe"
    val result: String = myLambda(name)
    println(result)

مثال: 

            
    val myLambda: (String , Int) -> String = {
        name , age ->
        "My name is $name, I'm $age years old, this is a message from lambda expmression"
    }

در مثال بالا عبارت لاندا دارای دو پارامتر است یکی برای اسم و دیگری برای سن

صدا زدن عبارت به صورت تابع: 

            
    val name = "Joe"
    val age = 24
    val result: String = myLambda(name, age)
    println(result)

توجه: name و age اسم های دلخواه برای پارامتر های تعریف شده هستند

برچسب زدن به عبارت لامبدا:

میتونیم به مقداری که برای عبارت تعریف میکنیم برچسب بزنیم تا هنگام بازگرداندن نتیجه به صورت آشکار مشخص کنیم میخواهیم نتیجه رو از داخل مقدار عبارت برگردونیم: 

            
    val myLambda: () -> String = mylambda@{
        return@mylambda "Hello From Lambda"
    }

برچسبmylambda@{...} اسم دلخواهیه که به عنوان برچسب برای عبارت تعریف میکنیم.

توابع بی نام ( anonymous functions ) :

همینطور که از اسمش مشخصه این توابع اسم ندارن نحوه ی بیان کردنشون مانند توابع معمولیه فرقشون با توابع معمولی نداشتن اسمه این توابع رو میتونیم به یک متغیر اختصاص بدیم و برای صدا زدن مانند عبارت لاندا عمل میکنیم

مثال:

تعریف تابع: 

            
    val myAnonymous = fun(){
        println("Hello From Anonymous")
    }

صدا زدن تابع: 

            
    myAnonymous()

مثال:

تعریف تابع: 

            
    val myAnonymous = fun(name: String): String{

        return "Hello $name from Anonymous"
    }

صدا زدن تابع:

            
    val name = "Joe"
    val message = myAnonymous(name)
    println(message)

مثال:

تعریف تابع: 

            
    val myAnonymous = fun(name: String, age: Int): String {

        return "Hello my name is $name, I'm $age years old this is a message from anonymous"

    }

صدا زدن تابع:

            
    val name = "Joe"
    val age = 24
    val message = myAnonymous(name , age)
    println(message)
